# 模板字段业务核对清单

核对日期：2026-07-16

## 怎么看这份清单

- 本清单按当前系统的实际处理方式整理，重点列出“**暂不影响最终判断**”的模板字段，供业务同事确认是否有遗漏的业务规则。
- **模板是否有用到**：回答该字段填写的内容会不会被当前系统拿来判断结果；“否（只留档）”表示内容不会参与当前判断。
- **现有用途**：说明即使不参与判断，字段内容目前仍会被系统怎样保存、展示或导出。
- “暂不影响最终判断”不代表字段会丢失：上传后仍会保存，在案例详情和导出的 Excel 中可以查看。
- 科技金融、绿色金融、数字金融、养老金融中，填写了内容的字段都会提供给系统作为判断依据；这四类模板没有单独列出的闲置字段。
- 模板中的字段名称不能随意删改；如果字段名称缺失、重复或无法识别，系统会提示模板不符合要求。字段内容可以为空，但空内容不会成为判断依据。
